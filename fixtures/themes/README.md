# Theme Scanner Fixtures

These fixtures preserve only the minimum paths and scheme keys needed to test
the theme scanner and inspection service. Raster and font files are textual
placeholders and are not valid renderable assets.

The fixture families come from `docs/THEME_COMPATIBILITY.md`:

- `baked-ui`
- `composited-grid`
- `static-composition`
- `scheme-only-partial`
- `empty-unsupported`

Do not expand these into copies of complete reference themes. Add only files
needed to protect a specific loading behavior.
